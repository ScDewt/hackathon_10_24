Репозиторий - https://github.com/ScDewt/hackathon_10_24

Изначально этот репозиторий - это заготовка для разработки сервисов на разных языках - PHP, Go, Python, Node.js и других.

Внутри репозитория есть соответствующие каталоги для этих языков. Внутри каталога каждого языка есть уже некоторые предустановленные библиотеки/фреймворки и сделан API метод /api/health, где проверяется подключение к БД.

Внутри docker-compose.yml ты найдешь порты, по которым опубликованы эти языки и их API.

Python

curl -XGET localhost:8080/api/health
curl -XGET http://hackathon.scdewt.ru:8080/api/health
http://hackathon.scdewt.ru:8080/api/health

PHP

curl -XGET localhost:8081/api/health
curl -XGET http://hackathon.scdewt.ru:8081/api/health
http://hackathon.scdewt.ru:8081/api/health

Go

curl -XGET http://hackathon.scdewt.ru:8082/api/health
http://hackathon.scdewt.ru:8082/api/health
